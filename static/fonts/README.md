# Fonts

`Optimistic-Display-*.woff2` are the **Optimistic Display** typeface, copied
from the React Native website repository:

- Source: `facebook/react-native-website`
  `website/static/fonts/Optimistic-Display-*.woff2`
- Commit: `a6a305451bf541ed994d51ca7fee220db1f642ef`

These files are used to match the React Native documentation theme. Optimistic
is Meta's brand typeface; the React Native website repository's MIT license
covers its source code. If Flux's distribution requirements change, swap these
for a clearly-licensed alternative and update the `@font-face` rules in
`src/css/react-native-theme.scss`.
